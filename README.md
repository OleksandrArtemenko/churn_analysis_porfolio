# Churn Analysis

This project includes an exploratory analysis of a subscription-based media company. It focuses on investigating the churn issue across available features and evaluating predictive analytics techniques using several machine learning models.

---

## Main Technologies Used
- **Pandas**
- **Matplotlib**
- **Seaborn**
- **scikit-learn**
- **statsmodels**

---

## Project Stages

### 1. Exploratory Data Analysis
- Initial exploration of numerical and categorical features  
- Analysis of customers and their behavior  
- Correlation analysis and investigation of differences between groups  
- Overview of churn across different dimensions  

### 2. Churn Prediction
- Preparing the data for training and testing machine learning models  
- Evaluating multiple models for churn prediction, including Logistic Regression, Decision Tree, Random Forest, and CatBoost  
- Comparing model performance using confusion matrices, tables, and visualizations  

### 3. Conclusions
- Final insights based on all stages of the analysis
